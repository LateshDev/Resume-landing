# 🚀 ResumeFlow – AI Resume Builder Landing Page

A modern and responsive landing page for *ResumeFlow*, an AI-inspired resume builder designed to help users create professional, ATS-friendly resumes with ease.

This project was built using *HTML5, CSS3, and JavaScript*, following modern web development practices including semantic HTML, responsive design, reusable CSS, and clean project organization.
